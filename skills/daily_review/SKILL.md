# daily_review

## Purpose
Produce a daily performance and operations review draft.

## Inputs
- `logs/daily_log.csv`
- `logs/expenses.csv`
- `logs/income.csv`
- `logs/projects.csv`
- `config/goals.md`

## Output
- `reports/daily/YYYY-MM-DD-daily-review.md`

## Required Sections
1. Daily Summary
2. Evidence of Progress
3. Drift/Risk Signals
4. Money Notes
5. Project Notes
6. Tomorrow’s Top 3 Actions
7. One System Improvement Suggestion

## Rules
- Show draft first.
- Save only after user approval.
