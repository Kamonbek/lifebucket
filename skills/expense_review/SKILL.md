# expense_review

## Purpose
Create a finance overview from recorded income and expenses.

## Inputs
- `logs/expenses.csv`
- `logs/income.csv`

## Output
- Finance summary draft

## Required Sections
1. Income
2. Expenses
3. Savings rate if possible
4. Burn rate if possible
5. Suspicious/waste categories
6. Next month limits

## Rules
- Do not invent missing values.
- Ask for missing data.
